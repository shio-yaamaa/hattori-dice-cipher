# JとRの配分

制限
- sum: 強制
- max: 強制
- min: 任意、ただしエラー
- ratio: 任意

sumを決定する際にmaxは考慮されてるので、sumとmaxは相反しない

J_quota + R_quota = sum
J_quota <= J_max
R_quota <= R_max
(J_quota >= J_min)
(R_quota >= R_min)
(R_quota / J_quota = 2～4)

sum: 常に考慮する
max: 最後に切り取る

minに届かない場合
J_min + R_min > sum
→minの割合に比例して配分
(それからmaxで切り取る)

else
R_quota / J_quota = 2～4のランダムにして、J_quotaをランダムで出す
min違反なら移しかえる

最後に
max違反→移しかえる
min違反→エラーをtrueに
